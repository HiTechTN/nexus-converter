Installation rapide

3 méthodes au choix. Choisissez celle qui vous convient.

⚡ Option 1 — Une commande (curl)

La méthode la plus rapide — une seule ligne de commande suffit :

# Télécharge + installe tout automatiquement
curl -fsSL https://github.com/HiTechTN/nexus-converter/releases/latest/download/install.sh | bash

Le script détecte votre OS, installe les prérequis (Node.js, pnpm, FFmpeg, yt-dlp), et configure tout automatiquement.

2 Démarrer l'application

cd ~/nexus-converter && ./start.sh

✨ L'application sera accessible sur http://localhost:3000

📦 Option 2 — Via Git (pour contributeurs)

# Cloner le dépôt
git clone https://github.com/HiTechTN/nexus-converter.git
cd nexus-converter

# Lancer l'installateur
./install.sh

# Démarrer
./start.sh

1 Cloner le dépôt

git clone https://github.com/HiTechTN/nexus-converter.git
cd nexus-converter

2 Lancer avec Docker Compose

Le Dockerfile multi-stage contient tout le nécessaire (Node.js, FFmpeg, yt-dlp).

# Construire et lancer
docker compose up --build

# Ou en arrière-plan
docker compose up --build -d

3 Accéder à l'application

🌐 http://localhost (port 80)

Le reverse proxy Nginx sert le frontend et proxy l'API.

1 Prérequis système

  • Node.js ≥ 18 — nodejs.org
  • pnpm ≥ 8 — npm install -g pnpm
  • FFmpeg — ffmpeg.org
  • yt-dlp — pip install yt-dlp
  • Python 3 ≥ 3.8 — python.org

2 Installer les dépendances

git clone https://github.com/HiTechTN/nexus-converter.git
cd nexus-converter
pnpm install

3 Builder les libs partagées

pnpm --filter @workspace/api-zod run build
pnpm --filter @workspace/api-client-react run build

4 Lancer (deux terminaux)

# Terminal 1 — API Server
PORT=3000 pnpm --filter @workspace/api-server run dev

# Terminal 2 — Frontend
PORT=5173 pnpm --filter @workspace/media-converter run dev

API Endpoints

Contrat OpenAPI 3.1 — Tous les endpoints de l'API REST

MéthodeRouteDescription
GET /api/healthz Health check (yt-dlp, FFmpeg, dossier tmp)
POST /api/media/info Récupérer les métadonnées d'une URL (titre, durée, formats disponibles)
POST /api/media/convert Démarrer une conversion à partir d'une URL
GET /api/media/progress/:jobId Flux SSE de progression en temps réel
GET /api/media/download/:jobId Télécharger le fichier converti
POST /api/media/upload Uploader un fichier local pour conversion
